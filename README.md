# Space_Shooter_Game
Top-down shooter demonstrating recoil physics, vector math, and real-time gameplay using Python and Pygame.

**Space_Shooter_Game** is a minimalist top-down arcade shooter built with **Python and Pygame**, featuring a unique recoil-based movement system. Instead of traditional WASD controls, the player navigates entirely through the physics of weapon recoil, creating a fast-paced and skill-driven gameplay experience.
